Установка имени и электронной почты
git config --global user.name "Your Name"
git config --global user.email "your_email@whatever.com"
Имя ветки по умолчанию
git config --global init.defaultBranch main
Корректная обработка окончаний строк
git config --global core.autocrlf true
git config --global core.safecrlf warn
Создание репозиторий
git init
Добавьть страницу в репозиторий
git add hello.html
git commit -m "Initial Commit"
Проверька состояние репозитория
git status
Коммит
git commit -m
переменная среды GIT_EDITOR
параметр конфигурации core.editor
переменная среды VISUAL
переменная среды EDITOR
Получение списка произведенных изменений
git log
Однострочная история
git log --pretty=oneline
--pretty="..." — определяет формат вывода
%h — укороченный хеш коммита
%ad — дата коммита
| — просто визуальный разделитель
%s — комментарий
%d — дополнения коммита («головы» веток или теги)
%an — имя автора
--date=short — сохраняет формат даты коротким и симпатичным
Создание тегов
git tag
Просмотр Тегов в логах
git log main --all
Переключитесь на ветку main
git switch main
Отмена изменений
git restore
Команда restore с флагом --staged очищает индекс.
